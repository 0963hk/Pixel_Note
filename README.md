# Pixel Note 1.0 用户手册

**文档版本**：v1.0

**最后更新**：2026-01-20

**适用系统**：Windows 10 / Windows 11

---

## 1. 产品概述

**Pixel Note** 是一款专为 Windows 平台设计的轻量级桌面便签应用程序。该软件摒弃了传统繁杂的界面设计，采用极简主义的复古像素风格。它旨在为用户提供一个纯净、无干扰的记录环境，支持快速记录灵感、待办事项及临时备忘。

### 主要特性

- **极简像素美学**：无系统原生边框，定制化的像素风格 UI。
- **轻量化运行**：单文件绿色执行，极低的内存占用。
- **高分屏适配**：内置 High DPI Awareness，确保在 2K/4K 屏幕上字体清晰锐利。
- **无干扰交互**：支持“窗口置顶”及隐形调整手柄，最大化书写区域。
- **多主题切换**：内置 5 种精心调配的复古配色方案。

## 2. 安装与运行 (Installation)

### 2.1 系统要求

- **操作系统**：Windows 10 (1903版本及以上) 或 Windows 11。
- **磁盘空间**：至少 20MB 可用空间。
- **依赖组件**：无（程序已集成运行环境）。

### 2.2 安装步骤

Pixel Note 采用便携式设计，无需经过传统的安装程序。

1. 下载 Pixel_Note_v1.exe 文件。
2. 将其放置在您希望保存的任意文件夹中（建议放置在非系统盘目录）。
3. 双击 Pixel_Note_v1.exe 即可启动。

### 2.3 卸载

由于软件不写入注册表，卸载时直接删除 Pixel_Note_v1.exe 文件及生成的配置文件（如有）即可。

## 3. 界面说明 (Interface)

软件主界面分为以下三个功能区域：

1. **标题栏 (Title Bar)**
    - 显示软件名称 "PIXEL NOTE"。
    - 右侧 `[X]` 按钮用于关闭程序。
    - 按住标题栏任意位置可拖动窗口位置。
2. **文本编辑区 (Text Area)**
    - 占据窗口主要面积，支持多行文本输入。
    - 采用等宽像素字体，支持自动换行。
3. **调整手柄 (Resize Area)**
    - 位于窗口**右下角**（隐形设计）。
    - 鼠标悬停于右下角 15x15 像素区域时，指针会自动变为双向箭头，提示可调整窗口大小。

---

## 4. 操作指南

### 4.1 基础操作

- **移动窗口**：鼠标左键按住顶部标题栏区域，拖动至桌面任意位置释放。
- **调整大小**：鼠标移动至窗口**右下角**边缘，当光标变为斜向箭头时，按住左键拖动以改变便签的长宽。
- **关闭程序**：点击标题栏右上角的 `[X]` 按钮，或在右键菜单中选择“Exit”。

### 4.2 个性化设置（右键菜单）

在程序界面的任意位置（标题栏或文本区）**单击鼠标右键**，可唤起功能菜单：

- **Theme (切换主题)**：
    - `Yellow`：经典便签黄（默认）。
    - `Green`：复古掌机绿。
    - `White`：极简纯白。
    - `Gray`：经典视窗灰。
    - `Black`：夜间模式/高对比度。
- **Always on Top (窗口置顶)**：
    - 勾选此项，便签将始终悬浮于所有其他应用程序窗口之上，防止被遮挡。
    - 再次点击可取消勾选。
